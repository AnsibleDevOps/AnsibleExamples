Introduction
============
``ansible-bootstrap`` is an Ansible role to prep a host in order to be managed
by Ansible. It will lock down SSH, install Fail2ban, set the root password, and
install some additional Python packages.

Installation
~~~~~~~~~~~~
.. code-block:: console
   git clone https://github.com/mrlesmithjr/ansible-bootstrap.yml
